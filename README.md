# GeoPandas GeoDataFrames for Mexican States and Municipalities

This repository contains Geopandas GeoDataFrames in the Parquet compressed format in the data directory:

* States in Mexico: data/mexico_admin1.parquet
* Municipalities in Mexico: data/mexico_admin2.parquet

The [notebook](notebooks/Download_Shapefile.ipynb) downloads Shapefiles for States (admin1) and Municipalities (admin2) from the 2020 Mexican Census: [INEGI](https://www.inegi.org.mx/temas/mg/).

Data [description](https://www.inegi.org.mx/app/biblioteca/ficha.html?upc=889463807469)
